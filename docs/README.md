## ✅ 예외 상황 정리
- 원하는 기능
  - 공백 입력
  - 기능에 포함되지 않는 입력
  - 현재 상태 (메인 화면, 경로 기준)에 적합하지 않은 입력
- 출발역
  - 공백 입력
  - 등록된 지하철에 없는 입력
- 도착역
  - 공백 입력
  - 등록된 지하철에 없는 입력
  - 출발역과 같은 입력
  - 출발역과 도착역이 연결되어 있지 않은 입력
